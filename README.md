# official-yijinji-archive

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内容以静态页面为主，不针对任何特定域名或网站，仅作为页面资料的存储与展示空间。

## 项目简介

本仓库旨在集中管理一批独立的 HTML 页面，方便查阅、引用或存档。每个页面均为自包含的 HTML 文件，无需额外依赖即可在浏览器中直接打开。仓库本身不提供任何形式的在线服务或内容导航，仅作为文件存储与版本管理用途。

## 目录说明

```
├── pages/          # 存放所有归档的 HTML 页面
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── README.md       # 本文件
└── LICENSE         # 仓库授权文件
```

- `pages/`：每个子目录或文件代表一个独立页面，命名方式尽量反映页面内容或用途。
- `assets/`：按需存放页面所需的共用资源，避免重复引用。
- `LICENSE`：本仓库使用 MIT 许可证，详情请参阅 LICENSE 文件。

## 页面归档说明

- 所有页面均为静态 HTML，不含后端逻辑或动态内容。
- 页面可能引用外部资源（如字体、图标库），但不保证外部资源长期可用。
- 部分页面可能包含示例数据或占位内容，仅供存档与演示目的。
- 仓库不保证页面的完整性、准确性或时效性，内容可能随时更新或移除。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器直接打开 `pages/` 目录下的任意 HTML 文件。
3. 也可通过本地 HTTP 服务器运行（如 Python 的 `http.server` 模块），以获得更完整的浏览体验。
